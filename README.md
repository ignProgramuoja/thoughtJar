# thoughtJar Lite

A lightweight, early-access snapshot of the thoughtJar project, built in Python using CustomTkinter.
thoughtJar Lite represents a minimal, stable reference version of the editor, showcasing core file handling and text editing functionality. It is feature-frozen and not intended for active development.
---
## Current features
- Create, open, save, and rename text files  
- Basic text editing:
  - Undo / Redo
  - Cut / Copy / Paste
  - Select All
- Keyboard shortcuts
- Custom menu system:
  - File
  - Edit
  - Format
  - View
  - More
---
## Not included (by design)
- Plugins
- Themes
- Markdown or syntax highlighting
- Autosave
- Settings persistence
---
## Project status
⚠️ **Feature-frozen / legacy build**
This version is no longer under active development.  
New features and experiments are implemented in newer branches and forks (e.g. 0.3a and experimental forks).
---
## Planned for the full version
- Plugin system
- Theming support
- Autosave and crash recovery
