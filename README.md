thoughtJar Lite

A lightweight, early-access text editor built in Python (CustomTkinter), intended as the foundation for a more advanced note-taking and writing application.
This Lite version focuses on core editor functionality only and serves as a testing ground for UI design, file handling, and editor behavior.

Current features
Create, open, save, rename text files
Basic text editing (undo, redo, cut, copy, paste, select all)
Keyboard shortcuts
Custom menu system (File / Edit / Format / View / More)

Not included (by design)
Plugins
Themes
Markdown or syntax highlighting
Autosave
Settings persistence
Project status
